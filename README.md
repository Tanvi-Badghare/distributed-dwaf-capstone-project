# Distributed Web Application Firewall (DWAF)

## Overview
A distributed, privacy-preserving WAF using ML-based detection, 
Zero-Knowledge Proofs for verification, Byzantine Fault Tolerant consensus,
and STIX/TAXII for threat sharing.
